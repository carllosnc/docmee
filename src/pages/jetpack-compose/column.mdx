# Column

## Overview

The `Column` composable is a fundamental layout component in Jetpack Compose that arranges its children vertically in a linear fashion. It's similar to `LinearLayout` with vertical orientation in the traditional Android View system. The `Column` is perfect for creating vertical lists, forms, and any UI that requires vertical arrangement of elements.

## Basic Syntax

```kotlin
Column(
    modifier: Modifier = Modifier,
    verticalArrangement: Arrangement.Vertical = Arrangement.Top,
    horizontalAlignment: Alignment.Horizontal = Alignment.Start,
    content: @Composable ColumnScope.() -> Unit
)
```

## Parameters

- **modifier**: Modifier to be applied to the Column
- **verticalArrangement**: Arrangement of children vertically within the Column
- **horizontalAlignment**: Default horizontal alignment for children
- **content**: The composable content inside the Column scope

## Code Examples

### 1. Basic Column with Text Elements

```kotlin
@Composable
fun BasicColumnExample() {
    Column(
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp)
    ) {
        Text(
            text = "First Item",
            fontSize = 18.sp,
            fontWeight = FontWeight.Bold
        )
        Text(
            text = "Second Item",
            fontSize = 16.sp,
            color = Color.Gray
        )
        Text(
            text = "Third Item",
            fontSize = 14.sp,
            style = TextStyle(textDecoration = TextDecoration.Underline)
        )
    }
}
```
*Creates a simple vertical list of text elements.*

```kotlin
fun ConditionalColumnExample() {

                    fontWeight = FontWeight.SemiBold
                )
                Text(
                    text = "This content appears when the button is pressed.",
                    color = Color.Gray
                )
                Text(
                    text = "You can add more detailed information here.",
                    fontSize = 12.sp
                )
            }
        }
    }
}
```
*Shows/hides content conditionally with animation.*

### 10. Animated Column Content

```kotlin
@Composable
fun AnimatedColumnExample() {
    var itemCount by remember { mutableStateOf(3) }
    
    Column(
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp),
        verticalArrangement = Arrangement.spacedBy(8.dp)
    ) {
        Row(
            modifier = Modifier.fillMaxWidth(),
            horizontalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            Button(
                onClick = { if (itemCount > 0) itemCount-- }
            ) {
                Text("Remove Item")
            }
            Button(
                onClick = { itemCount++ }
            ) {
                Text("Add Item")
            }
        }
        
        Text(
            text = "Items: $itemCount",
            fontWeight = FontWeight.Bold
        )
        
        repeat(itemCount) { index ->
            AnimatedVisibility(
                visible = true,
                enter = slideInVertically() + fadeIn(),
                exit = slideOutVertically() + fadeOut()
            ) {
                Card(
                    modifier = Modifier
                        .fillMaxWidth()
                        .animateContentSize(),
                    elevation = CardDefaults.cardElevation(defaultElevation = 2.dp)
                ) {
                    Text(
                        text = "Dynamic Item ${index + 1}",
                        modifier = Modifier.padding(16.dp)
                    )
                }
            }
        }
    }
}
```
*Demonstrates animated content addition and removal in a Column.*

## Vertical Arrangement Options

```kotlin
// Available vertical arrangements
Arrangement.Top                    // Children at the top
Arrangement.Bottom                 // Children at the bottom  
Arrangement.Center                 // Children centered
Arrangement.SpaceBetween          // Equal space between children
Arrangement.SpaceAround           // Equal space around children
Arrangement.SpaceEvenly           // Equal space between and around
Arrangement.spacedBy(8.dp)        // Fixed space between children
```

## Horizontal Alignment Options

```kotlin
// Available horizontal alignments in ColumnScope
Alignment.Start                   // Left alignment
Alignment.CenterHorizontally      // Center alignment
Alignment.End                     // Right alignment
```

## Best Practices

### 1. Performance Considerations
```kotlin
// Good: Use LazyColumn for long lists
LazyColumn {
    items(longList) { item ->
        ItemComposable(item)
    }
}

// Avoid: Regular Column with many items
Column {
    longList.forEach { item ->
        ItemComposable(item) // Performance issue for large lists
    }
}
```

### 2. Spacing Consistency
```kotlin
// Preferred: Use verticalArrangement for consistent spacing
Column(
    verticalArrangement = Arrangement.spacedBy(16.dp)
) {
    // Content
}

// Alternative: Manual spacing with Spacer
Column {
    Text("Item 1")
    Spacer(modifier = Modifier.height(16.dp))
    Text("Item 2")
}
```

### 3. Accessibility
```kotlin
Column(
    modifier = Modifier.semantics {
        contentDescription = "User profile information"
    }
) {
    Text("Name: John Doe")
    Text("Email: john@example.com")
}
```

3. **Settings Screens** - List of preferences and options
4. **Profile Screens** - User information in vertical layout
5. **Card Content** - Internal layout of cards and containers
6. **Navigation Drawers** - Menu items arranged vertically
