# Row

## Overview

The `Row` composable is a fundamental layout component in Jetpack Compose that arranges its children horizontally in a linear fashion. It's similar to `LinearLayout` with horizontal orientation in the traditional Android View system. The `Row` is perfect for creating horizontal lists, toolbars, button groups, and any UI that requires horizontal arrangement of elements.

## Basic Syntax

```kotlin
Row(
    modifier: Modifier = Modifier,
    horizontalArrangement: Arrangement.Horizontal = Arrangement.Start,
    verticalAlignment: Alignment.Vertical = Alignment.Top,
    content: @Composable RowScope.() -> Unit
)
```

## Parameters

- **modifier**: Modifier to be applied to the Row
- **horizontalArrangement**: Arrangement of children horizontally within the Row
- **verticalAlignment**: Default vertical alignment for children
- **content**: The composable content inside the Row scope

## Code Examples

### 1. Basic Row with Text Elements

```kotlin
@Composable
fun BasicRowExample() {
    Row(
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp),
        horizontalArrangement = Arrangement.spacedBy(12.dp)
    ) {
        Text(
            text = "First",
            fontSize = 16.sp,
            fontWeight = FontWeight.Bold,
            color = Color.Blue
        )
        Text(
            text = "Second",
            fontSize = 16.sp,
            color = Color.Green
        )
        Text(
            text = "Third",
            fontSize = 16.sp,
            style = TextStyle(textDecoration = TextDecoration.Underline)
        )
    }
}
```
*Creates a simple horizontal arrangement of text elements with spacing.*

### 4. Button Group with Row

```kotlin
@Composable
fun ButtonGroupExample() {
    var selectedButton by remember { mutableStateOf(0) }
    
    Row(
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp),
        horizontalArrangement = Arrangement.spacedBy(8.dp)
    ) {
        Button(
            onClick = { selectedButton = 0 },
            colors = ButtonDefaults.buttonColors(
                containerColor = if (selectedButton == 0) Color.Blue else Color.Gray
            ),
            modifier = Modifier.weight(1f)
        ) {
            Text("Option 1")
        }
        
        Button(
            onClick = { selectedButton = 1 },
            colors = ButtonDefaults.buttonColors(
                containerColor = if (selectedButton == 1) Color.Blue else Color.Gray
            ),
            modifier = Modifier.weight(1f)
        ) {
            Text("Option 2")
        }
        
        Button(
            onClick = { selectedButton = 2 },
            colors = ButtonDefaults.buttonColors(
                containerColor = if (selectedButton == 2) Color.Blue else Color.Gray
            ),
            modifier = Modifier.weight(1f)
        ) {
            Text("Option 3")
        }
    }
}
```
*Creates a button group with equal weight distribution.*

### 8. Progress Indicator Row

```kotlin
@Composable
fun ProgressRowExample() {
    var progress by remember { mutableStateOf(0.3f) }
    
    Column(
        modifier = Modifier.padding(16.dp),
        verticalArrangement = Arrangement.spacedBy(16.dp)
    ) {
        // Progress with labels
        Row(
            modifier = Modifier.fillMaxWidth(),
            verticalAlignment = Alignment.CenterVertically,
            horizontalArrangement = Arrangement.spacedBy(8.dp)
        ) {
            Text(
                text = "0%",
                fontSize = 12.sp,
                color = Color.Gray
            )
            LinearProgressIndicator(
                progress = progress,
                modifier = Modifier.weight(1f),
                color = Color.Blue,
                trackColor = Color.LightGray
            )
            Text(
                text = "100%",
                fontSize = 12.sp,
                color = Color.Gray
            )
        }
        
        // Progress value and controls
        Row(
            modifier = Modifier.fillMaxWidth(),
            verticalAlignment = Alignment.CenterVertically,
            horizontalArrangement = Arrangement.SpaceBetween
        ) {
            Text(
                text = "Progress: ${(progress * 100).toInt()}%",
                fontWeight = FontWeight.Medium
            )
            Row(
                horizontalArrangement = Arrangement.spacedBy(8.dp)
            ) {
                Button(
                    onClick = { progress = maxOf(0f, progress - 0.1f) }
                ) {
                    Text("-")
                }
                Button(
                    onClick = { progress = minOf(1f, progress + 0.1f) }
                ) {
                    Text("+")
                }
            }
        }
    }
}
```
*Shows progress indicator with controls in a Row layout.*

## Horizontal Arrangement Options

```kotlin
// Available horizontal arrangements
Arrangement.Start                 // Children at the start (left)
Arrangement.End                   // Children at the end (right)
Arrangement.Center                // Children centered
Arrangement.SpaceBetween          // Equal space between children
Arrangement.SpaceAround           // Equal space around children  
Arrangement.SpaceEvenly           // Equal space between and around
Arrangement.spacedBy(8.dp)        // Fixed space between children
```

## Vertical Alignment Options

```kotlin
// Available vertical alignments in RowScope
Alignment.Top                     // Top alignment
Alignment.CenterVertically        // Center alignment
Alignment.Bottom                  // Bottom alignment
Alignment.Baseline               // Baseline alignment for text
```

## Best Practices

### 1. Performance Considerations
```kotlin
// Good: Use LazyRow for long horizontal lists
LazyRow {
    items(longList) { item ->
        ItemComposable(item)
    }
}

// Avoid: Regular Row with many items
Row {
    longList.forEach { item ->
        ItemComposable(item) // Performance issue for large lists
    }
}
```

### 2. Responsive Design with Weight
```kotlin
// Responsive button layout
Row(modifier = Modifier.fillMaxWidth()) {
    Button(
        onClick = { /* Action 1 */ },
        modifier = Modifier.weight(1f)
    ) { Text("Button 1") }
    
    Spacer(modifier = Modifier.width(8.dp))
    
    Button(
        onClick = { /* Action 2 */ },
        modifier = Modifier.weight(1f)
    ) { Text("Button 2") }
}
```

### 3. Consistent Spacing
```kotlin
// Preferred: Use horizontalArrangement for consistent spacing
Row(
    horizontalArrangement = Arrangement.spacedBy(16.dp)
) {
    // Content
}

// Alternative: Manual spacing with Spacer
Row {
    Text("Item 1")
    Spacer(modifier = Modifier.width(16.dp))
    Text("Item 2")
}
```

### 4. Accessibility
```kotlin
Row(
    modifier = Modifier.semantics(mergeDescendants = true) {
        contentDescription = "User actions: edit, delete, share"
    }
) {
    IconButton(onClick = { /* Edit */ }) {
        Icon(Icons.Default.Edit, contentDescription = "Edit")
    }
    IconButton(onClick = { /* Delete */ }) {
        Icon(Icons.Default.Delete, contentDescription = "Delete")
    }
    IconButton(onClick = { /* Share */ }) {
        Icon(Icons.Default.Share, contentDescription = "Share")
    }
}
```

## Common Use Cases

1. **Toolbars and App Bars** - Navigation and action buttons
2. **Button Groups** - Multiple related actions
3. **Icon and Text Combinations** - Labels with icons
4. **Card Content** - Horizontal layout within cards
5. **Form Controls** - Input fields with labels or buttons
6. **Tab Bars** - Horizontal navigation tabs
7. **Progress Indicators** - Progress bars with labels
8. **Image Galleries** - Horizontal scrolling images

## Related Components

- **LazyRow** - For efficient horizontal scrolling lists
- **Column** - For vertical linear layouts
- **Box** - For overlapping and stacked layouts  
- **ConstraintLayout** - For complex positioning requirements
- **Scaffold** - For overall screen structure with app bars
