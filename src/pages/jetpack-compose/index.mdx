# Jetpack Compose

## Layout Components
- [**Column** - Arranges children vertically](./jetpack-compose/column)
- [**Row** - Arranges children horizontally](./jetpack-compose/row)
- [**Box** - Stacks children on top of each other](./jetpack-compose/box)
- [**ConstraintLayout** - Complex layouts with constraints](./jetpack-compose/constraint_layout)
- [**Scaffold** - Basic material design layout structure](./jetpack-compose/scaffold)
- [**LazyColumn** - Vertically scrolling list](./jetpack-compose/lazy-column)
- **LazyRow** - Horizontally scrolling list
- **LazyVerticalGrid** - Grid layout with vertical scrolling
- **Spacer** - Adds empty space between elements

## Text Components
- **Text** - Displays text
- **TextField** - Basic text input
- **OutlinedTextField** - Text input with outlined border
- **BasicTextField** - Minimal text input component

## Button Components
- **Button** - Standard material button
- **OutlinedButton** - Button with outlined border
- **TextButton** - Text-only button
- **IconButton** - Button containing an icon
- **FloatingActionButton** - Circular floating action button
- **ExtendedFloatingActionButton** - FAB with text and icon

## Selection Components
- **Checkbox** - Checkbox for boolean selection
- **RadioButton** - Radio button for single selection
- **Switch** - Toggle switch
- **Slider** - Range slider for numeric values
- **RangeSlider** - Dual-handle range slider

## Navigation Components
- **TopAppBar** - App bar at the top
- **BottomAppBar** - App bar at the bottom
- **NavigationBar** - Bottom navigation bar
- **NavigationRail** - Side navigation for larger screens
- **TabRow** - Horizontal tabs
- **ScrollableTabRow** - Scrollable tabs

## Media Components
- **Image** - Displays images
- **Icon** - Displays vector icons
- **AsyncImage** - Loads images from URLs (Coil library)

## Surface Components
- **Surface** - Material design surface
- **Card** - Material design card
- **ElevatedCard** - Card with elevation
- **OutlinedCard** - Card with outlined border

## Dialog Components
- **AlertDialog** - Standard alert dialog
- **BasicAlertDialog** - Minimal alert dialog
- **ModalBottomSheet** - Bottom sheet modal
- **DropdownMenu** - Dropdown menu

## Progress Components
- **LinearProgressIndicator** - Linear progress bar
- **CircularProgressIndicator** - Circular progress indicator

## Utility Components
- **Divider** - Visual separator line
- **Badge** - Small status indicator
- **Chip** - Compact element for tags/filters
- **Snackbar** - Brief message at bottom of screen
