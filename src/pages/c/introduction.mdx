## Introduction

C is a powerful general-purpose programming language created by **Dennis Ritchie** at Bell Labs in **1972**. It has influenced many modern languages and remains widely used for:

- Operating systems development
- Embedded systems
- System programming
- Game engines
- High-performance applications

### Why Learn C?

‚úÖ **Performance** - Direct hardware access and efficient execution  
‚úÖ **Portability** - Runs on virtually any platform  
‚úÖ **Foundation** - Understanding C helps you learn other languages  
‚úÖ **Control** - Full control over memory and system resources

## Getting Started

### Basic Program Structure

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

**Components:**
- `#include <stdio.h>` - Header file for input/output
- `int main()` - Entry point of every C program
- `printf()` - Function to print output
- `return 0` - Successful program termination

### Compiling and Running

```bash
# Compile
gcc program.c -o program

# Run
./program
```

### If-Else Statement


### Recursion

```c

    return n * factorial(n - 1);
```

## Pointers

### Basic Pointers

```c
int num = 42;
int *ptr = &num;  // ptr stores address of num

printf("Value: %d\n", num);           // 42
printf("Address: %p\n", &num);        // Memory address
printf("Pointer value: %p\n", ptr);   // Same address
printf("Dereferenced: %d\n", *ptr);   // 42
```

### Pointer Arithmetic

```c
int arr[5] = {10, 20, 30, 40, 50};
int *ptr = arr;

printf("%d\n", *ptr);       // 10
printf("%d\n", *(ptr + 1)); // 20
printf("%d\n", *(ptr + 2)); // 30
```

### Pointers and Functions

```c
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5, y = 10;
    swap(&x, &y);
    printf("x: %d, y: %d\n", x, y);  // x: 10, y: 5
    return 0;
}
```

### Pointer to Pointer

```c
int num = 42;
int *ptr = &num;
int **pptr = &ptr;

printf("%d\n", **pptr);  // 42
```

## Dynamic Memory

### malloc, calloc, realloc, free

```c
#include <stdlib.h>

int main() {
    // malloc - allocates memory
    int *arr = (int *)malloc(5 * sizeof(int));
    if (arr == NULL) {
        printf("Memory allocation failed\n");
        return 1;
    }
    
    // Use the memory
    for (int i = 0; i < 5; i++) {
        arr[i] = i * 10;
    }
    
    // realloc - resize memory
    arr = (int *)realloc(arr, 10 * sizeof(int));
    
    // free - deallocate memory
    free(arr);
    
    // calloc - allocates and initializes to zero
    int *arr2 = (int *)calloc(5, sizeof(int));
    free(arr2);
    
    return 0;
}
```

### Memory Leak Prevention

```c
// Always free allocated memory
int *ptr = (int *)malloc(sizeof(int));
if (ptr != NULL) {
    *ptr = 100;
    free(ptr);
    ptr = NULL;  // Good practice
}
```

## File Handling

### File Operations

```c
#include <stdio.h>

int main() {
    FILE *fp;
    
    // Writing
    fp = fopen("data.txt", "w");
    if (fp == NULL) {
        printf("Error opening file\n");
        return 1;
    }
    fprintf(fp, "Hello, File!\n");
    fclose(fp);
    
    // Reading
    fp = fopen("data.txt", "r");
    char buffer[100];
    while (fgets(buffer, 100, fp) != NULL) {
        printf("%s", buffer);
    }
    fclose(fp);
    
    // Appending
    fp = fopen("data.txt", "a");
    fprintf(fp, "Appended text\n");
    fclose(fp);
    
    return 0;
}
```

### File Modes

| Mode | Description |
|------|-------------|
| `"r"` | Read (file must exist) |
| `"w"` | Write (creates new or overwrites) |
| `"a"` | Append (adds to end) |
| `"r+"` | Read and write |
| `"w+"` | Write and read |
| `"a+"` | Append and read |

## Best Practices

### 1. Use Meaningful Names
```c
// Good
int studentAge;
float accountBalance;

// Bad
int x;
float a;
```

### 2. Check Return Values
```c
FILE *fp = fopen("file.txt", "r");
if (fp == NULL) {
    perror("Error opening file");
    return 1;
}
```

### 3. Always Free Memory
```c
int *ptr = malloc(sizeof(int));
// Use ptr
free(ptr);
ptr = NULL;
```

### 4. Initialize Variables
```c
int count = 0;
float price = 0.0;
char *name = NULL;
```

### 5. Use const for Constants
```c
const int MAX_USERS = 100;
const float TAX_RATE = 0.15;
```

## Common Pitfalls

‚ö†Ô∏è **Buffer Overflow**
```c
char buffer[10];
// Dangerous - can overflow
gets(buffer);  // DON'T USE

// Safe alternative
fgets(buffer, sizeof(buffer), stdin);
```

‚ö†Ô∏è **Memory Leaks**
```c
// Bad

// ... forgot to free

// Good
int *ptr = malloc(100);
// ... use ptr
free(ptr);
```

‚ö†Ô∏è **Dangling Pointers**
```c
int *ptr = malloc(sizeof(int));
free(ptr);
ptr = NULL;  // Prevent dangling pointer
```

‚ö†Ô∏è **Uninitialized Variables**
```c
int x;           // Bad - uninitialized
int y = 0;       // Good - initialized
```
